# Verificacion-DearthCraft
Esta es la verificacion de DearthCraft https://discord.gg/dearthcraft

# Instalacion
Edita el archivo `config.json` pon el *TOKEN* y donde quieres que envie el enbed

```json
{
  "token": "Tu token",

  "verificationchannel": "Chanel ID"
}
```
Iniciar bot

```text
npm i
node .
```
```text
node .
```
